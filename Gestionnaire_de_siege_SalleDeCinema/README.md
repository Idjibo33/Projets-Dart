# Système de Réservation de Siège de salle de cinéma

Petit projet en Dart simulant la gestion des places dans une salle de cinéma.

## Fonctionnalités
- Initialisation automatique de 10 sièges (A1 à A10).
- Utilisation d'une `Map` pour un accès instantané aux données.
- Système de réservation sécurisé avec gestion des **Exceptions**.

## Notions techniques utilisées
- Programmation Orientée Objet (Classes, Attributs, Méthodes).
- Collections (Map, Boucles).
- Gestion d'erreurs (Try-Catch, Throw).
