# Parcours d'apprentissage Dart - Portfolio

Bienvenue sur mon dépôt de projets **Dart**. Ce dossier regroupe mes exercices et projets réalisés pour maîtriser le langage Dart, de la logique de base aux concepts avancés.


## Compétences démontrées
- **Programmation Orientée Objet (POO)** : Classes, Héritage, Encapsulation.
- **Manipulation de Collections** : Utilisation avancée de `.where()`, `.map()`, `.sort()`.
- **Gestion des Erreurs** : Utilisation des blocs `try-catch` et des Exceptions.
- **Typage Fort** : Maîtrise de la Null-Safety et des Enums.
